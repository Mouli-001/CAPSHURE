CAPSHURE - Anti-Phishing & Fake CAPTCHA Detector
CAPSHURE is a lightweight Chrome extension (Manifest V3) designed to detect fake CAPTCHAs and scan suspicious links on any webpage using the VirusTotal API. It helps protect users from phishing attempts and malicious URLs by providing real-time visual alerts.

🚀 Features
Fake CAPTCHA Detection: Uses regex and iframe signatures to identify potential "human verification" pages used in phishing.

Automated URL Scanning: Automatically extracts and scans the first few links and scripts on a page using VirusTotal.

Visual Status Banners: * 

🟢 Green: Page is clean.

🟡 Yellow: API connection failed or quota exceeded.

🔴 Red: Malware or malicious links detected.

Toggle Control: Easily enable or disable protection via the popup menu.
